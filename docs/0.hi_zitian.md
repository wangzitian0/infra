- 把 SSH 私钥（可 sudo 的账号）存入仓库 Secrets：`VPS_SSH_KEY`。
- 在 Secrets 写好主机信息：`VPS_HOST`（必填）、`VPS_USER`(可空，默认 root)、`VPS_SSH_PORT`(可空，默认 22)。
- 决定 kube-apiserver 的对外域名：写到 `K3S_API_ENDPOINT`，并在 DNS 指向 VPS（否则证书校验会用 IP）。
- 如需固定版本，写 `K3S_VERSION`（例如 v1.28.7+k3s1）；否则保留空值跟随 `K3S_CHANNEL`。
- 在 GitHub Actions 手动触发一次 `Deploy k3s to VPS`，确认 kubeconfig artifact 能下载、`kubectl get nodes` 正常。
